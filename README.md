# Sorteador de Números

Este é um projeto que utilizei para praticar JavaScript e lógica de programação.

A estrutura em HTML e CSS já estava pronta, e minha parte foi desenvolver toda a lógica da aplicação usando JavaScript.

O objetivo do projeto é permitir que o usuário sorteie números aleatórios dentro de um intervalo definido.

---

## Funcionalidades implementadas

- Captura dos valores digitados pelo usuário
- Validação para garantir que o número mínimo seja menor que o máximo
- Geração de números aleatórios dentro do intervalo informado
- Garantia de que não haja números repetidos
- Exibição do resultado na tela
- Função para reiniciar os campos
- Controle de estado do botão "Reiniciar"

---

## Tecnologias utilizadas

- HTML (estrutura base)
- CSS (estilização)
- JavaScript (lógica desenvolvida por mim)
